ลูกค้า
  - เข้าสู่ระบบและสมัครเพื่อเข้าใช้งาน
  - มีฟีเจอร์การกรองหาโต๊ะที่ตามความต้องการของลูกค้า
  - แสดงรายละเอียดของแต่โต๊ะ
  - มีฟีเจอร์จองโต๊ะและระบุความต้องการพิเศษได้
  - แสดงรายการและสถานะการจอง

ร้านอาหาร
  - เข้าสู่ระบบเพื่อเข้าใช้งาน
  - ดูรายการจองที่เข้ามา
  - มีฟีเจอร์กรองเพื่อดูรายจองที่เข้ามาในแต่ละช่วงเวลาและแต่ละสถานะของการจองได้
  - ยืนยันและยกเลิกสถานะการจอง
  - มีdashboard แสดงจำนวนของสถานะรายการจอง
